# 百合花之王 妹子图片爬虫

## 说明

1.无需自行设置参数，通过采集相应栏目进行爬取

2.获取每个栏目的妹子个人链接

3.获取每个妹子的自拍合集

4.将这些图片分一级二级文件保存

5.通过json文件作为中转消息，可二次爬取

6.通过一个标记文件，进行中断再爬取设置

## 流程

1.先读取标记文件，若标记文件有被标记的内容说明上一次爬取过程有中断

2.通过标记文件，省去从头开始的弊端

3.获取了标记信息，开始爬取

4.采集对应的栏目，将每个栏目下的妹子地址保存为json文件

5.对栏目采集完成后，对每个栏目下的妹子图片开始爬取

6.每次爬取完一个妹子，将下标标记记录在标记文件中，可供二次爬取

7.爬取到的妹子图片各自保存在对应的文件夹下
